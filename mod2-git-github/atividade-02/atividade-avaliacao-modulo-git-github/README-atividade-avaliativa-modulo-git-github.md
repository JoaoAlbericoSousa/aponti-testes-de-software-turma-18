# Estratégia de Execução - Atividade Avaliação Módulo: Git e GitHub

Este repositório foi estruturado de forma a segmentar e isolar cada etapa da atividade prática de exploração do Markdown no GitHub.

## 🚀 Estratégia de Desenvolvimento

Para garantir a rastreabilidade e a organização do aprendizado, adota-se a seguinte estratégia:
1. **Exercícios Individualizados:** Cada exercício listado no roteiro será desenvolvido dentro de sua própria pasta dedicada.
2. **Histórico Isolado:** O isolamento em diretórios próprios evita conflitos de sintaxe e permite a evolução progressiva de um README simples para um profissional.
3. **Nomenclatura Padronizada:** As pastas seguem o padrão numérico e descritivo baseado na legenda da atividade para facilitar a correção.
